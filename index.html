jbcom/pkgs

Unified package repo: Homebrew, Scoop, and Chocolatey from one git tree.

Install

Homebrew

macOS, Linux, WSL2+Linuxbrew.

brew tap jbcom/tap
brew install <package>

Scoop

Windows.

scoop bucket add jbcom https://github.com/jbcom/pkgs
scoop install <package>

Chocolatey

Windows.

choco install <package>

How it works

Individual projects (e.g. radioactive-ralph) publish package manifests to this repo via their release pipeline. Homebrew formulas land in Formula/, Scoop manifests in bucket/, and Chocolatey package sources in choco/. CI validates every change on PR.
